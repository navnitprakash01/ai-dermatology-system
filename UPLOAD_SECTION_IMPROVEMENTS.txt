2. ADVANCED IMAGE QUALITY METRICS
───────────────────────────────────────────────────────────────────────────────

✨ NEW FEATURE: Real-Time Quality Analysis

Three Key Metrics:

1. BRIGHTNESS ☀️
   • Analyzes image lighting level
   • Target: >40% brightness
   • Prevents dark/underexposed images
   
2. CLARITY 🔍
   • Evaluates focus and sharpness
   • Target: >50% clarity
   • Ensures readable skin details
   
3. COVERAGE 📐
   • Checks affected area visibility
   • Target: >60% coverage
   • Verifies proper framing

Visual Indicators:
  • Progress bars for each metric
  • Color-coded status:
    - Green (>70%): Excellent
    - Purple (50-70%): Good
    - Red (<50%): Needs Improvement
  • Animated fills with staggered timing
  • Real-time updates as image loads

Quality Validation:
  • Automatic analysis after image upload
  • Smart threshold checking
  • User-friendly feedback messages
  • Actionable improvement guidance

7. USER EXPERIENCE FLOW
───────────────────────────────────────────────────────────────────────────────

Complete Journey:

1. Initial State
   ├─ Upload zone visible
   ├─ Step 1 highlighted (Upload)
   ├─ Helper text below header
   └─ Status: "Upload a clear photo of the affected area"

2. User Drags or Clicks
   ├─ Upload zone scales + background changes
   ├─ Upload icon rotates
   ├─ Preview state activates
   └─ Image file selected

3. Image Loads (0.6s delay for UX)
   ├─ Preview displays with animation
   ├─ Step 1 mark as complete (✓)
   ├─ Quality analysis begins
   └─ Step 2 becomes active (Eye icon)

4. Quality Analysis Complete
   ├─ Metrics displayed with animated bars
   ├─ File info card shows details
   ├─ Quality status badge appears
   ├─ Status updates: "Image quality is being validated"
   └─ Quality results shown

5. Quality Check (Two Paths)
   
   Path A: Quality GOOD
   ├─ Green badge: "Quality Good"
   ├─ All metric bars fill green
   ├─ Analyze button ENABLED
   ├─ Step 3 activated (Brain icon)
   ├─ No error message shown
   └─ Status: "Ready for AI analysis"
   
   Path B: Quality POOR
   ├─ Red badge: "Improve Quality"
   ├─ Metric bars show red areas
   ├─ Analyze button DISABLED
   ├─ Error message appears
   ├─ Helpful guidance displayed
   └─ User can Clear and retry

6. User Actions
   
   If Quality Good:
   └─ Click "Analyze Image" → Begin AI Analysis
   
   If Quality Poor:
   ├─ Click "Clear" → Return to upload zone
   └─ Upload better image
